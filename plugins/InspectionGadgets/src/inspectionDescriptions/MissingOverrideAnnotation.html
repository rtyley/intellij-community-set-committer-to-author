This inspection reports any methods which override methods in a superclass but do not have the @java.lang.Override annotation.

For safety, this inspection is disabled unless the project uses JDK 5.0.

Use the checkbox below to have this inspection ignore the java.lang.Object methods equals(), hashCode() and toString(). The risk of these methods going missing and thus your code not compiling anymore due to the @Override annotation is vanishingly small.
